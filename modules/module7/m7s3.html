Los 5 Por Qués: Profundizando

¿Qué es?

Técnica interrogativa iterativa que explora las relaciones causa-efecto subyacentes a un problema particular.

Cómo funciona

  1. Comenzar con el enunciado del problema (el "Efecto")
  2. Preguntar "¿Por qué?" está sucediendo y registrar la respuesta
  3. Si esa respuesta no revela la causa raíz, preguntar "¿Por qué?" nuevamente
  4. Repetir ~5 veces (o hasta identificar la causa raíz)

Beneficios

Simple de usar
Fomenta pensamiento más profundo
Ayuda a ir más allá de síntomas obvios

"El objetivo es llegar a una causa que, si se aborda, evitaría que el problema se repitiera."

Ejemplo Tecniseguros

Problema: Cliente recibió documento de póliza incorrecto
Respuesta 1: El agente seleccionó la plantilla incorrecta
Respuesta 2: El sistema de nombres de plantillas es confuso
Respuesta 3: No se estableció una convención de nomenclatura estándar
Posible Causa Raíz: El sistema se implementó apresuradamente sin una prueba completa de usabilidad Área de enfoque para la mejora

Evite culpar a las personas - busque fallas en el sistema o proceso
